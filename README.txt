Creates a manifiest file using the Provider Customization API of Windows Live Writer to create a custom sidepanel in Windows Live Writer for your blog

== Description ==

Using the [Provider Customization API of Windows Live Writer](http://msdn2.microsoft.com/en-us/library/bb463266.aspx) this adds a link to the head section of your blog, which links to a manifest file.  This tells Windows Live Writer to display custom values in the sidepanel for your blog.  Currently this will show your favicon if it exists & a watermark image & a link to the comments section in your admin and a link to a stats page set through the options page

== Installation ==

1. Download the zip file & extract the contents to /wp-content/plugins
2. Activate the plugin through the 'Plugins' menu in WordPress
4. (optional) Open the options page to set the location of the stats page and the watermark image to use
